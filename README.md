# AI Interview Coach

## Overview
An AI-powered platform that helps users practice interviews and receive real-time feedback on their responses.

## Features
- Generates interview questions dynamically
- Evaluates user responses using AI
- Provides feedback on clarity, accuracy, and confidence
- Tracks performance across sessions

## Tech Stack
- Backend: Python (Flask/Django)
- AI Integration: OpenAI / Gemini API
- Database: SQL
- Frontend: HTML, CSS, JavaScript

## How It Works
1. User starts an interview session
2. System generates questions
3. User submits answers
4. AI evaluates and provides feedback
